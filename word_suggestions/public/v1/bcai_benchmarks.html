Known Limitations in v1 + V2 Roadmap

⚠ Hyphenated Compound Words

sixth-generation, ultra-reliable, low-latency, machine-type — returns 0 suggestions after the hyphen character.

V2 Fix: Compound word dictionary + hyphen-aware lookup layer

⚠ Numeric Tokens

6G, 5G, 2030, 27000 km/h — not in NLTK corpus. Returns 0.

V2 Fix: Alphanumeric + numeric lookup extension

⚠ Bracket + Mixed Tokens

(6G), (eMBB), (URLLC), (mMTC) — special characters not in matrix. Returns 0.

V2 Fix: Symbol-stripped fallback lookup

⚠ Acronyms in Caps

eMBB, URLLC, mMTC, ISAC — acronyms not in NLTK corpus. Returns 0.

V2 Fix: Acronym expansion dictionary layer

⚠ Inflected / Plural Forms

systems, represents, bridges, introduced — returns 0 after full inflected form typed.

V2 Fix: Morphological stemming + inflection lookup

⚠ Punctuation-Attached Words

unified, digital, distributed, — returns 0 when trailing comma/period attached.

V1.1 Fix: Strip trailing punctuation before lookup (quick fix)

V2 Roadmap — Expected Coverage Improvement

Post-v2 fixes projected to raise word coverage from ~72% to ~99%+
